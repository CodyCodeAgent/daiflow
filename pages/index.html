功能特性

为开发者打造的 AI 工作台

不是单次对话，而是完整的可追踪开发流水线。状态持久化、可重入、可审查。

🏠

完全本地运行

代码与数据不离开你的机器，无需上传到任何云端服务。AI 引擎在进程内运行，零网络依赖。

⚡

五阶段完整工作流

Init → Plan → Todo → Coding → Review，每个阶段都有 AI 辅助与对话能力，状态机严格管控流转。

📡

实时流式 AI 交互

WebSocket 单连接多路复用，AI 执行过程实时推送。支持 Session 中断恢复，重启不丢进度。

🛠️

全栈功能开发友好

关联前端、后端多个 Git 仓库，AI 同步理解跨仓结构与 API 边界，一次任务贯通全栈改动。

🔌

可插拔 Runner 后端

支持 Cody SDK（进程内）、Claude Code CLI、Cursor IDE 三种 Runner，三级解析策略灵活配置。

🧠

技能中心自进化

内置技能库持续沉淀项目知识，AI 读取 Skill 文件自动理解架构规范，新任务无需重复解释上下文。

工作流

五阶段 DevFlow

从任务初始化到代码合并，每个阶段清晰可见，都支持与 AI 实时对话。

阶段 01
🚀

任务初始化

拉取代码、检出分支、同步技能库，建立项目知识四层流水线（结构 / 业务流 / 模块 / 索引）。

阶段 02
📐

技术方案

AI 生成结构化技术方案，支持多轮对话调整。锁定后自动触发任务拆解，方案持久化为 plan.md。

阶段 03
📋

任务拆解

AI 将技术方案拆解为有序 Todo 列表，复用方案 Session 保持上下文连贯，支持对话微调。

阶段 04
💻

编码实现

逐条 Todo 独立 AI Session 执行，实时 Diff 展示，支持跳过、重试与批量执行，每步可审查。

阶段 05
🔍

代码审查

汇总全量 Diff，AI 生成 Commit Message，支持对话式代码审查，一键提交 MR 到 Git。

Runner 后端

选择你的 AI 引擎

统一的 AbstractAgentRunner 协议，支持多种 AI 执行后端，按任务 / 项目 / 全局三级配置。

🤖

Cody SDK

默认推荐。进程内运行 Cody AI，无需外部服务依赖，低延迟，支持多种 LLM 模型配置。

进程内 低延迟 pip install cody-ai
🧠

Claude Code CLI

集成 Anthropic Claude Code CLI，通过子进程调用，享受 Claude 的强大代码理解与生成能力。

CLI 子进程 Anthropic

Cursor IDE

集成 Cursor IDE，利用已有的 IDE 环境与配置，无缝融入现有开发工作流。

快速开始

三步启动 DaiFlow

无需复杂配置，几分钟内完成安装并开始你的第一个 AI 辅助开发任务。

bash — 快速安装
# 1. 克隆并安装
$ git clone https://github.com/CodyCodeAgent/daiflow && cd daiflow
$ python -m venv .venv && source .venv/bin/activate
$ pip install -r requirements.txt -e .

# 2. 构建前端
$ cd frontend && npm install && npm run build && cd ..

# 3. 启动（自动打开浏览器）
$ daiflow start

# 或使用 Electron 桌面版
$ cd electron && npm install && npm run dev
开始使用
